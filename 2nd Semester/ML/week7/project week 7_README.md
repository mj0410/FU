## How To Run

1. Download data 'GSE68086_TEP_data_matrix.txt.gz' from [NCBI GEO](https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE68086) and
   supplementary file 1 from [Zhang's paper](https://www.oncotarget.com/article/20903/text/#T1)
2. Run the codes with given dataset and [DeepSVR](https://github.com/griffithlab/DeepSVR) from week 6 (for multi-class classification)
3. Evaluate classification
