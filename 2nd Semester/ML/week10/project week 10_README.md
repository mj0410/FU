## How To Run

1. Download files of class 4 and 5 from [E.coli benchmark dataset](http://www.cac.science.ru.nl/research/data/ecoli/)
3. Run ML_week10_FeatureExtraction.ipynb with files from 1
    - read .mzXML files
    - smoothing
    - filterinf by MS level
    - feature extraction
    - restructure data using Tidy Data Framework
4. Run ML_week10_preprocessing.ipynb
    - merge all samples
    - remove rows(features) which have more than 5 missing values
    - knn imputation to handle missing values
    - select top 100 important features
5. Run ML_week10_classification.ipynb
    - feature binning
    - running Naive Bayes and Decision Tree
